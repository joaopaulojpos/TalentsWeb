# Talents API

### Versão
1.0.0

### Configuração

Importar base de dados id4854326_db_talents.sql

Instalar Dependências do SlimFramework 


```sh
$ composer install
```
### API Endpoints
```sh
$ http://localhost/talentsweb/api/public/api/profissional/login
$ http://localhost/talentsweb/api/public/api/profissionais
$ http://localhost/talentsweb/api/public/api/profissional/vagas?cd_profissional=
$ http://localhost/talentsweb/api/public/api/profissional/salvar
$ http://localhost/talentsweb/api/public/api/empresa/login
$ http://localhost/talentsweb/api/public/api/empresas
$ http://localhost/talentsweb/api/public/api/empresa/salvar
$ http://localhost/talentsweb/api/public/api/empresa/{id}
$ http://localhost/talentsweb/api/public/api/empresa/{id}/vagas
$ http://localhost/talentsweb/api/public/api/empresa/match
$ http://localhost/talentsweb/api/public/api/vaga/publicar
$ http://localhost/talentsweb/api/public/api/vaga/curtirVaga
$ http://localhost/talentsweb/api/public/api/vagas
$ http://localhost/talentsweb/api/public/api/cargos
$ http://localhost/talentsweb/api/public/api/cursos
$ http://localhost/talentsweb/api/public/api/idiomas

$ http://localhost/talentsweb/api/public/api/competencias_comport
```
